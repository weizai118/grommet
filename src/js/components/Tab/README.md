## Tab
One tab within Tabs.

## Usage

```javascript
import { Tab } from 'grommet';
<Tab />
```

## Properties

**plain**

Whether this is a plain tab with no style.

```
boolean
```

**title**

The title of the tab.

```
string
node
```
  
## Intrinsic element

```
button
```
## Theme
  
**global.colors.text**

The text color used inside the component. Expects `string | { dark: string, light: string }`.

Defaults to

```
{ dark: '#f8f8f8', light: '#444444' }
```
